I study how predictive screening algorithms and precision prevention strategies can be implemented in health systems — and how their clinical and economic value can be evaluated using real-world data. My dissertation, in partnership with Kaiser Permanente Northwest, examines EHR-embedded CKD risk prediction tools through the lens of implementation science and health technology assessment.

This portfolio demonstrates the analytic pipeline that connects **clinical problem identification → evidence synthesis → statistical analysis → economic evaluation → strategic decision-making**.
